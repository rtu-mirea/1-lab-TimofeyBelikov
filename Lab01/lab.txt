Лабораторная работа 1
Тема. Обработка массивов.
Пакеты java.io, java.lang. Классы Arrays, Math, System, Scanner.

Задание 1. Разработать программу, которая обрабатывает данные, хранящиеся в одномерном массиве, используя классический перебор элементов. 
Требования
1.	Разработать основную программу  - один класс с методом main и выполнить:
1.1.	Разработать методы, общие для всех вариантов: 
•	ввод массива с клавиатуры и датчиком случайных чисел класса Math.
•	вывод массива в направлении слева направо методом println в цикле for , без использования индекса;
•	вывод массива в направлении справа налево.
1.2.	Разработать дополнительно методы, указанные в варианте. В методах поиска предусмотреть параметры, принимающие значения (или значения поиска).
1.3.	Вывод результатов дополнительных методов выполнить в методе main.
1.4.	Разработать метод, реализующий консольный интерфейс управления программой, позволяющий продемонстрировать выполнение всех разработанных методов.
1.5.	Выполнить отладку программу.
1.6.	Выполнить тестирование программы.
2.	Внести изменения в структуру программы
2.1.	Создать класс по обработке массива с заданным набором  операций, переместив  в него методы, подготовленные в пункте 1 и необходимые переменные.
2.2.	Выполнить программу, используя экземпляр созданного класса.
Задание 2. Реализовать задание 1, используя методы классов: Arrays, Math, System, Random.

Личный вариант: 1 (•	Определить количество чисел массива принадлежащих заданному диапазону.
                   •	Упорядочить элементы массива по возрастанию методом прямого выбора    )
